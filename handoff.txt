4. LONG AUDIO HANDLING
   - OpenAI API upload limit is 25MB per request.
   - NEVER slice WebM/MP4 audio by raw bytes.
   - For long audio (>25MB), use time-based chunking on decoded audio
     (e.g. WAV / PCM mono 16kHz).
   - Avoid corrupted container uploads.

     - ADA.log must be verbose.
     - Debug/test-only features must be visible.
     - Errors must be logged clearly.
